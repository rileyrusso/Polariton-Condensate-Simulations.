# Polariton-Condensate-Simulations.
**Quantum Fluid &amp; Polariton Dynamics Simulator**  A Python-based suite (NumPy/Matplotlib) modeling driven-dissipative quantum fluids. Simulates Bogoliubov dispersion, roton softening, and LHY quantum pressure corrections. Includes Monte Carlo robustness testing and hardware constraint modeling cavity leakage/thermal decay for experimental viability
